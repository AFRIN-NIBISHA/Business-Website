# Business Website - Franklin Builders

This is a premium, modern, and fully responsive business showcase website for **Franklin Builders & Engineering Services** (Contractors in House Construction, Renovations, Interior Woodworks, and Commercial Buildings).

## 🚀 Tech Stack
- **Frontend**: React (Vite template)
- **Styling**: Tailwind CSS v4 (with custom CSS variables for light/dark theme switching)
- **Icons**: Lucide React
- **Lead Capture**: Seamless click-to-chat WhatsApp integrations on contact forms and service CTAs

## 🛠️ Features
- **Dynamic Scrollspy**: Highlights active page sections inside the sticky glassmorphic navigation bar on scroll.
- **Scroll Reveal Animations**: Sections smoothly slide and scale into view as you scroll.
- **Image Gallery Lightbox**: Categorized filterable gallery showing building phases and final facades with modal image zoom.
- **Full Theme Support**: Easily toggle between premium slate dark mode and clean slate light mode.

## 💻 Local Development
To run this project locally:
1. Navigate into the client directory:
   ```bash
   cd Client
   ```
2. Install npm dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production hosting (Netlify / Vercel):
   ```bash
   npm run build
   ```
